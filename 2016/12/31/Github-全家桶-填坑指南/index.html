##Github Desktop

###下载
亚马逊AWS国际是被墙的,你需要打开 小飞机右键 >> 系统代理模式 >> 全局模式
待你的小飞机 变色 之后,下载就会走代理(默认好像走的系统代理)
下载完成后,一定要关闭全局模式 因为很多程序会调用系统IE的
举个栗子 QQ音乐,QQ游戏

###代理设置
安装好之后,双击桌面的 Git Shell 运行如下两条命令

我更改过默认的端口,所以我填的是2333! 一般都是默认端口1080
